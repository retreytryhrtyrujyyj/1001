# 电商后台管理系统

## 项目简介
电商后台电商后台管理系统是针对电商系统的后台管理系统，分用户模块，商品模块，订单模块，物流信息模块等功能模块，电商后台管理系统是针对电商系统的后台管理系统，采用当前最流行的前后端分离的开发模式,后端使用Springboot框架，前端采用VUE框架，UI采用ElementUI框架。
